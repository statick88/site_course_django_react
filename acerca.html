Objetivo General del Curso

Este curso está diseñado para desarrolladores web principiantes o intermedios con conocimientos básicos de programación con Python, comprensión general de HTML y CSS; que deseen adquirir habilidades avanzadas en el desarrollo de aplicaciones web utilizando Django y React.

Módulos del Curso

  • Modulo 0: Bienvenida
  • Modulo 1: Python Básico
  • Modulo 2: Python Intermedio
  • Modulo 3: Python Avanzado
  • Modulo 4: Git y Github / Docker
  • Modulo 5: Django - Fundamentos
  • Modulo 6: Django - Desarrollo Intermedio / Desarrollo Avanzado
  • Modulo 7: HTML5, CSS3, JavaScript
  • Modulo 8: Node.js y NPM
  • Modulo 9: React - Fundamentos / Intermedio
  • Modulo 10: React - Desarrollo Avanzado
  • Modulo 11: Proyecto Final Django

Detalles del Curso

Duración: 6 semanas

Sesiones por semana: 3 sesiones de 2 horas

Días: Lunes, Miércoles y Viernes

Horario: 20:00 a 22:00
