Introducción a Python Básico.

Objetivo

  • Comprender los conceptos básicos de Python.
  • Familiarizarse con la sintaxis y la estructura del lenguaje.
  • Establecer las bases para el desarrollo de programas en Python.

Ejemplo 1: Hola Mundo en Python.


print("Hola Mundo")

El primer programa de cualquier lenguaje de programación es el famoso "Hola Mundo".

Ejemplo 2: Operaciones básicas


a = 10 // asignación de variable a
b = 5 // asignación de variable b
print(a + b) // suma
print(a - b) // resta
print(a * b) // multiplicación
print(a / b) // división

Python es un lenguaje de programación interpretado, interactivo y orientado a objetos.

Ejemplo 3: Estructuras de control


a = 10
b = 5
if a > b:
	print("a es mayor que b")
else:
	print("b es mayor que a")

Python soporta estructuras de control como if, else, elif, for, while, break, continue, etc.

Ejemplo 4: Bucle For


for i in range(5):
	print(i)

Python soporta bucles for y while.

Ejemplo 5: Funciones


def suma(a, b):
	return a + b
	print(suma(10, 5))

Python soporta funciones, clases, módulos, excepciones, etc.

Asignación

Desarrollar un programa en Python que solicite al usuario dos números y realice las cuatro operaciones básicas (suma, resta, multiplicación, división) con estos números.
