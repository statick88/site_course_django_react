¿Qué es Python? 💡

  • Python es un lenguaje de programación de alto nivel, interpretado y de propósito general.

    • Introducción 💡

    Python es un lenguaje de programación versátil y fácil de aprender.

    Ejemplo 💡

    Python se destaca por su sintaxis clara y legible. Por ejemplo, el clásico "Hola, mundo" en Python se escribe así:

    print("Hola, mundo")

    Ejercicio💡

    Escribe un programa en Python que imprima tu nombre en la pantalla.

    Sintaxis Básica💡

    La sintaxis en Python se refiere a la forma en que se escriben las instrucciones.

    Ejemplo💡

    Python usa sangrías (indentación) para definir bloques de código. Por ejemplo, en un bucle for:

    						
for i in range(5):
		print(i)

    Ejercicio💡

    Escribe un programa que calcule la suma de los números del 1 al 10.

    Variables y Tipos de Datos💡

    Las variables se utilizan para almacenar datos en la memoria durante la ejecución del programa.

    Ejemplo💡

    Declaración de variables en Python:

    						
nombre = "Juan"
edad = 25
altura = 1.75

    Ejercicio💡

    Crea variables para almacenar tu edad, nombre y estatura, e imprímelas en pantalla.

    Estructura de Control💡

    Las estructuras de control permiten tomar decisiones y repetir acciones en función de ciertas condiciones.

    Ejemplo💡

    Uso del condicional if:

    						
edad = 20
		if edad >= 18:
print("Eres mayor de edad")
		else:
print("Eres menor de edad")

    Ejercicio💡

    Escribe un programa que determine si un número ingresado por el usuario es par o impar.

    Funciones💡

    Las funciones son bloques de código reutilizables que realizan una tarea específica.

    Ejemplo💡

    Definición de una función en Python:

    						
def saludar():
		print("¡Hola!")

    Ejercicio💡

    Crea una función que calcule el área de un círculo dado su radio.

    Definición y Llamada a Funciones💡

    Para utilizar una función, primero debemos definirla y luego llamarla en nuestro código.

    Ejemplo💡

    Llamada a la función saludar() que definimos anteriormente: 

    						
saludar()

    Ejercicio💡

    Define una función que reciba dos números como argumentos y devuelva su suma.

    Parámetros y Argumentos💡

    Los parámetros son variables que se utilizan en la definición de una función, mientras que los argumentos son los valores que se pasan a la función cuando se llama.

    Ejemplo💡

    Función con parámetros:

    						
def saludar(nombre):
		print("¡Hola,", nombre, "!")

    Ejercicio💡

    Modifica la función que calcula el área del círculo para que reciba el radio como parámetro.
