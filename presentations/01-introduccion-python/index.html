¿Qué es Python? 💡

  • Python es un lenguaje de programación de alto nivel, interpretado y de propósito general.
  • Es conocido por su sintaxis clara y legible, lo que lo hace ideal para principiantes y profesionales por igual.

    • Variables y Tipos de Datos 🔢

  • En Python, las variables se pueden asignar directamente sin necesidad de declarar su tipo.
  • Los tipos de datos incluyen enteros, flotantes, cadenas, listas, tuplas y diccionarios.

    • Operadores ➕ ➖ ✖️ ➗

  • Python admite una amplia gama de operadores para realizar operaciones aritméticas, de comparación y lógicas.

    Ejemplo 1: Hola Mundo en Python 🐍.

    
print("Hola Mundo")

    El primer programa de cualquier lenguaje de programación es el famoso "Hola Mundo".

    Ejemplo 2: Operaciones básicas ➕ ➖ ✖️ ➗

    
a = 10 // asignación de variable a
b = 5 // asignación de variable b
print(a + b) // suma
print(a - b) // resta
print(a * b) // multiplicación
print(a / b) // división

    Python es un lenguaje de programación interpretado, interactivo y orientado a objetos.

    Ejemplo 3: Estructuras de control 📥 📤

    
a = 10
b = 5
if a > b:
	print("a es mayor que b")
else:
	print("b es mayor que a")

    Python soporta estructuras de control como if, else, elif, for, while, break, continue, etc.

    Ejemplo 4: Bucle For 🛠️

    
for i in range(5):
	print(i)

    Python soporta bucles for y while.

    Ejemplo 5: Funciones ⚙️

    
def suma(a, b):
	return a + b
	print(suma(10, 5))

    Python soporta funciones, clases, módulos, excepciones, etc.
