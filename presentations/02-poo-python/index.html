Programación Orientada a Objetos (POO) en Python 🐍

🎯 Objetivos:

  • Comprender los conceptos básicos del paradigma de la POO en Python.
  • Aplicar el concepto de clases y objetos en la resolución de problemas.
  • Explorar la encapsulación, herencia y polimorfismo como mecanismos clave de la POO.

Conceptos Básicos 💻

POO es un paradigma de programación que se basa en la creación de clases y objetos que interactúan entre sí para modelar y resolver problemas del mundo real de manera más efectiva.

En Python, la POO es una parte fundamental y poderosa del lenguaje, lo que permite la creación de código modular, reutilizable y fácil de mantener.

Clases y Objetos 💡

Clases 💡

Una clase es una plantilla que define las propiedades y comportamientos comunes de un tipo de objeto.

Define la estructura de los objetos, pero no contiene datos reales. En Python, las clases se definen utilizando la palabra clave class.

Ejemplo 💡


class Coche:
    # Constructor
    def __init__(self, marca, modelo):
        self.marca = marca
        self.modelo = modelo

    # Método
    def conducir(self):
        print(f"Conduciendo el {self.marca} {self.modelo}")

Objeto 💡

Un objeto es una instancia específica de una clase.

Se crea a partir de una clase y tiene sus propios valores para las propiedades de la clase.

Los objetos pueden acceder a los métodos y atributos definidos en la clase.

Ejemplo💡


class Persona:
    # Variable de clase
    especie = "Humano"

    # Constructor
    def __init__(self, nombre, edad):
        # Variables de instancia
        self.nombre = nombre
        self.edad = edad

    # Método
    def saludar(self):
        print(f"Hola, mi nombre es {self.nombre} y tengo {self.edad} años")

Atributos y Métodos 👧

Atributos: Los atributos son variables asociadas a un objeto que representan características o datos de ese objeto.

Pueden ser variables de instancia (pertenece a cada objeto) o variables de clase (compartidas por todos los objetos de la misma clase).

Ejemplo💡


class Persona:
    # Variable de clase
    especie = "Humano"

    # Constructor
    def __init__(self, nombre, edad):
        # Variables de instancia
        self.nombre = nombre
        self.edad = edad

    # Método
    def saludar(self):
        print(f"Hola, mi nombre es {self.nombre} y tengo {self.edad} años")

Metodos 📦

Métodos: Los métodos son funciones definidas en una clase que representan comportamientos o acciones que los objetos de esa clase pueden realizar.

Pueden acceder y modificar los atributos de la clase.

Encapsulación📦

Es el proceso de ocultar los detalles internos de un objeto y exponer solo las operaciones necesarias para interactuar con él.

En Python, la encapsulación se logra utilizando convenciones de nomenclatura y métodos especiales como getters y setters.

Herencia: 📦

Es un mecanismo que permite a una clase heredar atributos y métodos de otra clase.

La clase que hereda se llama subclase o clase hija, y la clase de la que se hereda se llama superclase o clase padre.

Polimorfismo 📦

Es la capacidad de un objeto para tomar varias formas.

En Python, el polimorfismo se logra mediante el uso de métodos con el mismo nombre en diferentes clases, que pueden comportarse de manera diferente según la clase a la que pertenezcan.
