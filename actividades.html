Bienvenido a la página de actividades del curso. Aquí encontrarás una lista completa de todas las asignaciones piloto por módulo que forman parte de nuestro programa de aprendizaje. Cada asignación está diseñada para proporcionarte una experiencia práctica y aplicada en los diferentes temas cubiertos en el curso.


Explora nuestras asignaciones desde los fundamentos de Python hasta el desarrollo avanzado de aplicaciones web con Django y React. Además, encontrarás tareas relacionadas con herramientas esenciales como Git y GitHub, así como tecnologías de desarrollo front-end como HTML5, CSS3 y JavaScript.


¡Sumérgete en el mundo del desarrollo de software y potencia tus habilidades con nuestras desafiantes y emocionantes asignaciones!
